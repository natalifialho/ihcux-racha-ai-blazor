# RachaAí Blazor 💸

Projeto desenvolvido em **Blazor .NET** para a disciplina de **IHC e UX**.

O sistema simula um aplicativo de divisão de gastos, permitindo visualizar:

- Total a receber
- Total a pagar
- Saldo geral
- Lista de grupos financeiros

## 🧩 Componentização

O projeto foi dividido em componentes reutilizáveis, como o `GrupoCard.razor`, responsável por exibir:

- Nome do grupo
- Categoria
- Valor formatado
- Indicadores visuais positivos e negativos

---

## 🎨 UX e Hierarquia Visual

Foram aplicados conceitos de:

- Hierarquia visual
- Responsividade
- Feedback visual
- Consistência
- Visibilidade do status do sistema

As cores verde e vermelha ajudam o usuário a identificar rapidamente valores positivos e negativos.

---

## ⚡ Maior Desafio

O maior desafio foi componentizar o `GrupoCard`, deixando o componente reutilizável e responsivo com estilização dinâmica.
